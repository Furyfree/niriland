# IntelliJ Remaps

Jeg udfylder denne fil ud fra dine screenshots.

| Gør | Ny keybind | Fjerner / erstatter |
| --- | --- | --- |
| Project in Tool Windows | `Ctrl+Shift+E` | Fjerner `Alt+1`; `Remove` valgt, så `Ctrl+Shift+E` fjernes fra `Recent Locations` i `Main Menu | View | View Recent Actions Group` |
| Commit in Tool Windows | `Ctrl+K` | Fjerner `Alt+0`; `Keep` valgt, så eksisterende `Ctrl+K`-bindings beholdes også |
| Terminal in Tool Windows | `Ctrl+Shift+T` | Fjerner `Alt+F12`; `Remove` valgt, så `Ctrl+Shift+T` fjernes fra `New Tab` i `Plugins | Terminal`, `Go to Test` i `Main Menu | Navigate | Goto by Reference Actions` og `Show Services Tree` i `Other` |
| Git in Tool Windows | `Ctrl+Shift+G` | Fjerner `Alt+9`; ingen konflikt vist i dialogen |
| AI Chat in Tool Windows | `Alt+Shift+J` | Fjerner `Alt+Shift+1`; `Remove` valgt, så `Alt+Shift+J` fjernes fra `Unselect Previous Occurrence` i `Database | Data Editor` og `Unselect Occurrence` i `Main Menu | Edit | Find` |
| Settings... | `Ctrl+,` | Fjerner `Ctrl+Alt+S`; ingen konflikt vist i dialogen |
| New Tab | `Ctrl+T` | `Remove` valgt, så `Ctrl+T` fjernes fra `Update Project` i `Main Menu | Git | VCS Group` |
| Update Project | `Alt+Shift+U` | Mistede `Ctrl+T` til `New Tab`; ingen konflikt på `Alt+Shift+U` |
| Go to Test | Ingen endnu | Mistede `Ctrl+Shift+T` til `Terminal in Tool Windows` |
| Show Services Tree | Ingen endnu | Mistede `Ctrl+Shift+T` til `Terminal in Tool Windows` |
